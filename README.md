# Tittle: Worldly Dictionary

This project is a class assignment for student at Moringa school. It  helps student grasp the concept of using HTML,CSS and JavaSCript to build Single Page Application (SPA). This project aims to improve user experience by providing interactive dictionary feature which allows user to search for words, view their definations, and save their favorite terms for future reference in one single page.
# Tools

1. HTML
2. JavaSCript
3. CSS

# Installation

1. Get to the GitHub repository https://github.com/Limotiz/Summative-Lab-Single-Page-Application

2. Fork the repository to your GitHub account.

3. Clone the forked repository to your local machine.

4. Navigate to the project directory

5. Run any other installation to install the necessary dependancies
